Postavljanje MySQL baze podataka

Nightsparrow zahtjeva MySQL bazu podataka -- većina dijeljenih web hosting paketa imaju mogućnost da napravite jednu.

Ako vaš web hosting koristi cPanel, evo kako možete napraviti bazu podataka:

1. Kada se prijavite u cPanel, nađite opciju MySQL® Database Wizard pod Databases i kliknite na nju. Ovo će otvoriti čarobnjak za stvaranje databaze.

2. U prvom koraku, stvarate databazu. U polje New Database: unesite ime svoje baze. Imajte na umu da se puno ime vaše databaze sastoji i od dijela koji se nalazi ispred upisanog. Dakle, ako vam se ispred prostora za upis pojavljuje pulsir_ i unesete baza kao ime baze, puno ime vaše baze podataka će biti pulsir_baza. Kada ste gotovi, kliknite na Create Database. Zapišite što ste upisali.

3. U drugom koraku, stvaramo korisnika za databazu. Pod Username: unesite ime korisnika za svoju bazu. Imajte na umu da se puno ime korisnika sastoji i od dijela koji se nalazi ispred upisanog. Dakle, ako vam se ispred prostora za upis pojavljuje pulsir_ i unesete ivica kao ime korisnika, puno ime vašeg korisnika za bazu podataka će biti pulsir_ivica. U iduće polje, Password: unesite lozinku za svog korisnika, te ju ponovite u sljedećem polju. Kada ste gotovi, kliknite na Create User. Zapišite što ste upisali.

4. U trećem koraku, potrebno je odrediti ovlasti za MySQL korisnika. Ako niste upoznati sa SQL privilegijama, označite ALL PRIVILEGES i kliknite na Next Step. Ako razumijete kako funkcioniraju SQL privilegije, dodajte koje su vam potrebne i nastavite.

5. Databaza je sad stvorena. Podatke koje ste unijeli u čarobnjak sada unesite u instalacijski program. Također je potrebno unijeti MySQL server kojem Nightsparrow će pristupati. Ako ovo ne znate, pokušajte unijeti localhost. Ako to ne uspije, upitajte svog pružatelja web hostinga za informacije o MySQL serveru.

Ako budete imali problema s nekim od koraka kod dodavanja databaze ili vaš pružatelj web hostinga ne koristi cPanel, molimo da se obratite vašem web hosting provideru.
